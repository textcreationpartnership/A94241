1. In this Schedule is contained the Excise, set and to be set upon severall Commodities, as well Forreign as Native, as hereafter followeth, viz.

    _ Sugars Vocat.

    _ Drugges.

    _ Vpholstery.

    _ Saltery.

    _ Soape.

    _ Paper.

    _ Hides, Skinnes and Leather.

    _ Glasses.

    _ Home or Native Commodities
